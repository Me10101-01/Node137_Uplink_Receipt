#### Overview

The Collective Consciousness Printing Press (CCPP) is a fan-based academic paper access tool that aggregates scholarly research aligned with the Legion of Minds Council's invention portfolio. It generates professionally formatted PDF compilations of research papers organized by theme.

#### Features

- **16 Curated Academic Papers** spanning:
  - LLM Forensics (INV-047)
  - AI Behavioral Analysis (INV-054, INV-055)
  - Quantum Multi-Agent Systems (INV-056)
  - DSL & Compiler Infrastructure (FlameLang)

- **Professional PDF Generation** with:
  - Title page with metadata
  - Invention alignment registry
  - Themed paper sections
  - Provenance attestation
  - Custom styling and formatting

#### Usage

```bash
# Install dependencies
pip install -r requirements.txt

# Generate PDF
python3 scripts/ccpp_generator.py
```

The PDF will be generated in the `receipts/` directory with a timestamped filename:
```
receipts/CCPP_Academic_Papers_YYYY-MM-DD_HHMMSS.pdf
```

#### Output Structure

1. **Title Page** - CCPP branding and metadata
2. **Invention Alignment Registry** - Quick reference table
3. **Themed Sections:**
   - LLM Forensics (4 papers)
   - AI Behavioral Analysis (4 papers)
   - Quantum Multi-Agent Systems (5 papers)
   - DSL & Compiler Infrastructure (3 papers)
4. **Provenance Attestation** - Legal and attribution notice

#### Dependencies

- `reportlab>=4.0.0` - Professional PDF generation library
